# HealthBridge Telemedicine Platform

HealthBridge is a comprehensive telemedicine platform that connects patients with healthcare providers through secure video consultations, digital prescriptions, and appointment management.

## Features

### 1. User Authentication
- Login/Signup system for patients and doctors
- Persistent login with localStorage
- Role-based access (patient/doctor)

### 2. Patient Dashboard
- Overview with health statistics
- Appointment management (upcoming/past)
- Digital prescriptions viewer
- Medical history tracking
- Profile management

### 3. Video Consultation
- Simulated video call interface
- Audio/video controls
- Real-time consultation notes
- Prescription issuance
- Participant management

### 4. Appointment Booking
- Doctor selection by specialty
- Date and time scheduling
- Reason for consultation
- Appointment status tracking

### 5. Additional Features
- Responsive design for all devices
- Testimonial slider
- Notification system
- Mobile-friendly navigation
- Smooth scrolling animations
